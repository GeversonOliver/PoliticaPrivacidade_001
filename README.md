# Política de Privacidade

Este repositório contém a Política de Privacidade do aplicativo **Revilo Cérebro Acadêmico**.

## Sobre o Aplicativo

O aplicativo foi desenvolvido com foco em auxílio acadêmico e não coleta dados pessoais dos usuários.

## Coleta de Dados

O aplicativo:

- Não coleta informações pessoais;
- Não exige cadastro;
- Não compartilha dados com terceiros;
- Não utiliza rastreamento de usuários;
- Não utiliza anúncios;
- Não utiliza serviços de analytics.

## Segurança

Mesmo sem coleta de dados, o aplicativo segue boas práticas de desenvolvimento para garantir uma experiência segura aos usuários.
